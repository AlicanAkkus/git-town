High-level command-line interface for Git

Git Town is an open-source Git plugin that provides high-level Git commands to make feature-branch based development more efficient and safe.

Install

A workflow layer on top of vanilla Git

Git is an intentionally generic and basic tool. It supports many different ways of using it equally well. It is a wonderful foundation for source code management.

Using it properly as a developer is cumbersome and repetitive, though.
As an example, safely merging a finished feature branch requires up to
15 individual Git commands!

Powerful commands

Git Town adds a number of high-level Git commands that automate typical operations in the software development workflow.

$ git hack

Cuts a new up-to-date feature branch off the main branch.

$ git sync

Updates your current branch with all ongoing changes.

$ git new-pull-request

Create a new pull request on GitHub or BitBucket for your current branch.

$ git ship

Delivers your completed feature branch and removes it.

FAQ

Does this force me into any conventions for my branches or commits?

Not at all. Git Town doesn’t require or enforce any particular naming convention or branch setup, and works with a wide variety of Git branching models and workflows.

How is this different from Git Flow?

Git Flow is a Git extension that provides specific and opinionated support for the powerful Git branching model with the same name. It doesn’t care too much about how you keep your work in sync with the rest of the team.

Git Town doesn’t care much about which branching model you use. It makes you and your team more productive and keeps your Git repo clean.

It is possible (and encouraged) to use the two tools together.

Which Git branching models are supported by Git Town?

Git Town is so generic that it supports all the branching models that we are aware of, for example GitHub Flow, GitLab Flow, Git Flow, and even working straight off the master branch.

Is it compatible with my other Git tools?

Yes, we try to be good citizens. If you run into any issues with your setup, please let us know!
